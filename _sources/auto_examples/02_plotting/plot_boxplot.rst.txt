Boxplot example
===============





.. image:: /auto_examples/02_plotting/images/sphx_glr_plot_boxplot_001.png
    :align: center





.. code-block:: python


    import pandas as pd
    import seaborn as sns
    import numpy as np

    conds = ['A','B','C','D','E','F','G']

    data = [np.random.random_integers(0,100,100).tolist() for cond in conds]

    data_df = pd.DataFrame(data)
    data_df=data_df.transpose()
    data_df.columns = conds

    sns.boxplot(data_df)
